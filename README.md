# Aerocode - Gestão de Produção de Aeronaves ✈️

Este é um sistema de linha de comando (CLI) desenvolvido em TypeScript para simular e gerenciar o processo de produção de aeronaves. O sistema conta com persistência de dados em arquivos de texto e diferentes níveis de permissão de acesso.

## 🛠️ Pré-requisitos

Certifique-se de ter o Node.js instalado na sua máquina. 

## 🚀 Como rodar o projeto

1. Abra o terminal na pasta raiz do projeto e instale as dependências do Node (como o `readline-sync` e o `typescript`):
```bash
npm install

-- Após a instalação, inicie o sistema rodando o arquivo principal:

npx ts-node src/index.ts
